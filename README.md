## Description

[Nest](https://github.com/nestjs/nest) Backend for the cars GBH project.

## Project setup

```bash
$ yarn install
```

## Prepare Environment variables

```bash
$ cp exemple_env .env
```

## Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev


## Run tests

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
